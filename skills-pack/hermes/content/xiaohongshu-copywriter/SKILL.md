# 小红书爆款文案生成器

## 触发条件

### ✅ 匹配关键词（满足任一即触发）
- 平台明确：小红书文案/笔记/种草文/小红书爆款/小红书风格
- 品类+内容：美妆笔记/穿搭分享/护肤心得/美食探店/好物推荐
- 动作需求：帮我写小红书/生成小红书/写篇种草/探店文案
- 打卡类：打卡攻略/拍照圣地/出片/宝藏店铺/私藏好物

### ❌ 排除关键词（明确不触发）
- 公众号/微信推文 → 用 wechat-article
- 抖音/快手/短视频脚本 → 用 short-video-script
- 朋友圈文案/朋友圈图文 → 拒绝
- 微博文案/微博长文 → 拒绝

### 🎯 上下文条件
- 用户提供产品/店铺信息 → 直接进入生成
- 用户只说"写小红书" → 询问品类+主题
- 用户提供截图 → 描述图片提取信息
- 用户指定风格 → 按要求生成

## 核心流程（6步）

### Step 1: 信息收集（缺什么问什么）
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| category | string | 待询问 | 美妆/穿搭/美食/探店/护肤/母婴/家居/数码 |
| topic | string | 待询问 | 具体产品/主题/场景 |
| audience | string | 学生党/职场人 | 目标人群 |
| style | string | 种草安利 | recommend/tutorial/avoid/mood/funny |
| tone | string | casual | casual/professional |

### Step 2: 标题生成（5选1展示）
按公式生成5个候选标题，每个含emoji：

| 公式 | 结构 | 示例 |
|------|------|------|
| 数字+痛点 | "N个[痛点]，你中了第X个" | "花了3000块才搞懂！这5个护肤误区你全中了" |
| 身份+成果 | "[身份]必看！[成果]分享" | "打工人必看！月入8k也能穿出高级感" |
| 对比冲突 | "[A] vs [B]，差距在于..." | "同样是租房，她家像豪宅我家像仓库" |
| 悬念好奇 | "[悬念]！[效果/感受]" | "闺蜜追着我问链接！这个平替也太绝了" |
| 场景+方案 | "[场景痛点]？这套直接封神" | "约会不知道穿什么？这套直接封神" |

### Step 3: 正文生成

**结构模板**
```
【开头Hook - 1-2句，20-40字】
制造共鸣或悬念，让用户停下划屏

【核心内容 - 3-5个要点】
每点用emoji引导，配合具体细节

【行动引导 - 1句，15-25字】
引导收藏/点赞/评论

【标签区 - 15个标签】
#标签1 #标签2 #标签3...
```

**写作规则**
- 必须用第一人称"我"
- 必须含emoji（每段至少1个）
- 口语化表达，像真人分享
- 具体数字/细节增加可信度

### Step 4: 标签生成（15个）

| 组别 | 数量 | 类型 | 示例 |
|------|------|------|------|
| 流量标签 | 3个 | 大热词 | #小红书爆款 #种草 #好物分享 |
| 精准标签 | 7个 | 品类词 | #精华液 #美白 #平价护肤 |
| 长尾标签 | 5个 | 细分词 | #学生党精华 #熬夜党护肤 |

### Step 5: 质量校验
- [ ] 标题≤20字且含emoji
- [ ] 正文用第一人称"我"
- [ ] 至少3个emoji
- [ ] 标签≥15个（流量+精准+长尾）
- [ ] 敏感词过滤（禁用：震惊/必看/千万别/后悔）

### Step 6: 输出交付

```

1️⃣ [标题1]
2️⃣ [标题2]
3️⃣ [标题3]
4️⃣ [标题4]
5️⃣ [标题5]

📌 选定标题：[用户选择后展示]

[正文内容]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🏷️ 标签：
#流量1 #流量2 #流量3 #精准1 #精准2...（15个）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 标签分类策略

### 流量标签（蹭热度）
- 平台词：小红书爆款、种草、宝藏、好物分享
- 趋势词：2024、必入、闭眼入、封神
- 情绪词：绝了、哭死、尖叫、疯狂安利

### 精准标签（触达目标）
- 品类词：精华液、面霜、口红、穿搭
- 功效词：美白、抗老、保湿、遮瑕
- 人群词：学生党、打工人、宝妈、干皮

### 长尾标签（被搜索）
- 细分词：平价替代、早C晚A、油痘肌
- 场景词：通勤穿搭、约会妆容、办公室零食
- 品牌词：（具体品牌）

## 边界约束

1. **字数限制**：正文100-1000字（300-500最佳）
2. **内容安全**：禁止医疗虚假宣传/金融诈骗/色情低俗
3. **版权要求**：不直接复制他人原创
4. **平台规则**：遵守小红书社区规范
5. **违禁词**：震惊/必看/千万别/后悔/不看后悔
